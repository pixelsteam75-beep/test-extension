# Quick Installation Guide

## How to Load the Extension in Chrome

1. **Open Chrome Extensions Page**
   ```
   chrome://extensions/
   ```
   Or: Menu (⋮) → More Tools → Extensions

2. **Enable Developer Mode**
   - Toggle the switch in the top-right corner

3. **Load Unpacked Extension**
   - Click "Load unpacked" button
   - Select the `extension` folder (the one containing manifest.json)

4. **Verify**
   - The "Wellness MVP Tracker" should appear in your extensions list
   - Click the extension icon in the toolbar to start using it

## First Use

- Click the extension icon to see the Pomodoro timer
- Right-click the icon → "Options" to view the dashboard
- Start browsing and the extension will automatically track your time

That's it! The extension is now running and tracking your activity locally.
